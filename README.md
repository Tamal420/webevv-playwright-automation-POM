# WebEVV Playwright Automation (POM)

## 📌 Project Overview

This project is a Playwright automation framework built using the Page Object Model (POM) design pattern.

## 🛠️ Tech Stack

* Python
* Playwright
* Pytest

## 📂 Project Structure

```
├── pages/          # Page Object classes
├── tests/          # Test cases
├── assets/         # Test data / files
├── utils.py        # Utility functions
├── config.py       # Configuration settings
├── conftest.py     # Pytest fixtures
├── requirements.txt
```

## 🚀 Setup Instructions

### 1. Clone the repo

```
git clone <your-repo-url>
cd webevv-playwright-automation-POM
```

### 2. Install dependencies

```
pip install -r requirements.txt
playwright install
```

### 3. Run tests

```
pytest
```

## 📊 Reports

After execution:

* Open `report.html`
